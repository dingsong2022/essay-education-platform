📝 논술 평가 플랫폼 - 완성 현황 정리
🎯 프로젝트 개요
한국 학생들을 위한 영어 논술 학습 및 평가 플랫폼

학생: 논술 작성 → AI 평가 → 개인 대시보드
교사: 전체 학생 관리 및 분석 대시보드
AI 챗봇: 소크라테스식 학습 도우미


✅ 1단계: 기본 논술 평가 시스템 (100% 완료)
🔐 사용자 인증 시스템

학생 회원가입/로그인: SHA-256 해시화, 중복 검사
교사 로그인: '교사정보' 시트 기반 인증 (보안 강화)
세션 관리: Streamlit 세션 상태로 로그인 유지

📝 논술 작성 시스템

주제 설정: 직접 입력 또는 8개 예시 주제 선택
작성 인터페이스: 실시간 단어/글자 수 카운터
영어 논술 가이드: 구조화된 작성 팁 제공

🤖 AI 평가 시스템 (Gemini 1.5 Flash)

5개 영역 평가: Content, Organization, Language, Coherence, Task Achievement
각 영역 20점씩 총 100점 만점
한국어 피드백: 상세한 개선점과 문법 수정 제안
점수 자동 추출: "총점:" 패턴 인식

💾 데이터 저장 (Google Sheets)

'사용자정보' 시트: 학생 가입 정보
'교사정보' 시트: 교사 계정 관리 (보안)
'논술데이터' 시트: 모든 논술문, 점수, 피드백 저장


✅ 2단계: AI 학습 도우미 (100% 완료)
🤖 소크라테스식 챗봇

질문 중심 대화: 답을 주지 않고 학생 스스로 생각하도록 유도
제한적 힌트: 필요시 간단한 아이디어나 예시만 제공
주제 관련성 체크: 설정된 주제와 무관한 질문은 거부
대화 히스토리: 연속된 맥락 유지
빠른 질문: 4가지 자주 묻는 질문 버튼


✅ 교사 대시보드 (100% 완료)
📊 4개 분석 탭

성과 분석:

점수 분포 히스토그램 (Plotly)
A-F 등급별 원형 차트
주제별 평균 점수 막대그래프


학생별 현황:

학생 성과 요약 테이블 (정렬 가능)
개별 학생 점수 추이 그래프


최근 제출물:

논술문 및 AI 피드백 조회
슬라이더로 표시 개수 조정


추이 분석:

월별 반 평균 점수 변화
주간 논술 제출량 추이



📈 전체 반 현황 지표

전체 학생 수, 총 논술 수, 반 평균 점수
활동 학생 수 (최근 30일 기준)


✅ 학생 개인 대시보드 (100% 완료)
📈 성과 추적

통계 카드: 총 작성 수, 평균 점수, 최고 점수, 최근 향상도
작성 목록: 검색, 정렬, 상세보기, 등급 표시
